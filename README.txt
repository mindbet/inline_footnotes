INTRODUCTION
------------
This module adds an Inline Footnotes plugin for Drupal's CKEditor.

INSTALLATION
------------
1. Add the Inline Footnotes module to your Drupal installation (versions 8.8+ supported).
2. Enable the Inline Footnotes module at /admin/modules, or using Drush.
3. Configure your WYSIWYG toolbar to include the button at /admin/config/content/formats
4. You must use the Full HTML format to use the plugin.
5. In the Toolbar Configuration, drag the Inline Footnotes icon to the Active toolbar, then Save Configuration.
6. You will then be able to add inline footnotes to your pages when you edit them in CKEditor

REQUIREMENTS
------------
CKEditor Module (Core)
